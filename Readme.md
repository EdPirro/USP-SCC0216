Como Usar:
----------

Para compilar simplesmente use 
```shell
    > make
``` 
, tendo o [Makefile](Makefile).
Para rodar, use:
```shell
    > make run 
```

Para mais informações sobre funcionamento do programa e suas entradas veja o [Relatório](Relatorio.pdf) do mesmo.
